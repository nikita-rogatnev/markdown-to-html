# INSTALLATION

### Requirements
    - install Node JS >= 7.6
    - install *npm* (if it isn't provided with Node JS)
    - install *gulp* globally using npm including "-g" flag
```
$ npm install -g gulp
```

### Install npm dependencies
```
$ npm install
```

### Build Commands

Kick off the build process:
```
$ gulp
```
